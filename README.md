# firefox-dictionary-extension

## Overview
This is a simple tool for Windows 11 that lets you change image file formats easily. Uses tkinterdnd2 & Pillow. 
Supports: JPEG, PNG, BMP, GIF, TIFF, WebP, and ICO.
